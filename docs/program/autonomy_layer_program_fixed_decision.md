# Universal Autonomy Layer - Зафиксированное решение

## Рамка решения

- **Технически:** Simulation-first с готовностью к переходу на реального робота.
- **Коммерчески:** Paid Discovery + Pilot-as-a-Product.
- **Команда:** старт 2 человека (`Autonomy` + `Platform`), 3-й как `BD/PM` по мере загрузки.
- **Цель на 8-10 недель:** рабочий демонстратор + выход на первый платный пилот.

## План-график (12 недель)

- Sprint 1 (недели 1-2): Foundation
- Sprint 2 (недели 3-4): Orchestration v1
- Sprint 3 (недели 5-6): Reliability v1
- Sprint 4 (недели 7-8): Pilot Packaging
- Sprint 5-6 (недели 9-12): Pre-pilot Hardening

## Стоимостные вилки

### Lean (1-2 человека)

- Горизонт: 3 месяца
- Оценка: **~$18k-$45k**
- Фокус: simulation-first + 1 целевой сценарий
- Ограничения: меньше покрытия тестами и slower hardening

### Base (2-3 человека)

- Горизонт: 3 месяца
- Оценка: **~$45k-$90k**
- Добавляет:
  - более глубокое тестовое покрытие
  - более сильную observability
  - более быстрый выход к пилоту

## Коммерческий оффер клиенту

### Paid Discovery

- Стоимость: **EUR 5k-15k**
- Длительность: 2-4 недели
- Результат: согласованные KPI/сценарий пилота и технический план исполнения

### Pilot-as-a-Product

- Стоимость: **EUR 25k-60k**
- Длительность: 8-12 недель
- Формат: KPI фиксируются в контракте

## One-pager оффера (каркас)

### Название

`Universal Autonomy Layer Pilot`

### Для кого

- Лаборатории
- Стандартизированные внутрискладские процессы

### Что решаем

- Снижение ручных вмешательств
- Рост надежности repeatable-операций

### Что входит (8-12 недель)

- Orchestration v1 для 1 сценария
- Fault handling + safe-state policy
- Telemetry + reliability dashboard
- API-интеграция v1 (базовый адаптер)
- Финальный reliability report + roadmap масштабирования

### KPI пилота

- Mission success rate (порог согласуется заранее)
- MTTR при отказах
- Доля авто-восстановлений
- Количество ручных вмешательств на N миссий

### Результат клиента

- Рабочий pilot-контур с измеримой надежностью
- Понимание TCO/ROI масштабирования
- План production rollout
